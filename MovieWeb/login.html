Movie Recommendation System:
About US

In this project movie recommendation system, recommendation techniques are very important in the fields of E-commerce and other Web-based services. This recommendation system also helps the new users to get a list of movies recommended to them based on the language, nationality and the age they belong to. One of the main difficulties is dynamically providing high-quality recommendation on sparse data. Here, a novel dynamic personalized recommendation algorithm is proposed, in which information contained in both ratings and profile contents are utilized by exploring latent relations between ratings, a set of dynamic features are designed to describe user preferences in multiple phases, and finally a recommendation is made by adaptively weighting the features. Experimental results on public datasets using existing systems show that the proposed algorithm has satisfying performance. In traditional Root Mean Square Error (RMSE) evaluations training and testing data are randomly sampled and the train and test split is not based on time.
